# SEO-документы для трех сайтов

В проекте подготовлено первичное SEO для трех отдельных лендингов в нише аварийного вскрытия замков. Для каждого сайта создан отдельный файл с семантикой, мета-тегами, структурой заголовков, SEO-текстом, FAQ, alt/aria-рекомендациями, Schema.org и базовым on-page ТЗ.

## Файлы

- `seo-klyuchnik-rf.md` — SEO для сайта `klyuchnik.rf`.
- `seo-master-zamkov.md` — SEO для сайта `master-zamkov.ru`.
- `seo-zamok-pro.md` — SEO для сайта `zamok.pro`.

## Краткое позиционирование

### `klyuchnik.rf`

Акцент на доверие, официальный подход, проверку права доступа, быстрый выезд 15-30 минут, аккуратное вскрытие без повреждений и оплату после результата.

Основной SEO-фокус: `вскрытие замков Москва`, `вскрытие замков Санкт-Петербург`, `аварийное вскрытие замков`, `открыть замок без ключа`, `вскрытие авто`, `вскрытие сейфа`.

### `master-zamkov.ru`

Акцент на практичный вызов мастера по замкам: быстро, понятно, круглосуточно, без лишней сложности для пользователя.

Основной SEO-фокус: `мастер по замкам Москва`, `мастер по замкам Санкт-Петербург`, `вызвать мастера по замкам`, `аварийный мастер замочник`, `замена замков`.

### `zamok.pro`

Акцент на экспертность и профессиональный подход: сложные замки, сейфы, офисы, официальный выезд, неразрушающие методы.

Основной SEO-фокус: `профессиональное вскрытие замков`, `вскрытие сложных замков`, `вскрытие сейфов Москва`, `вскрытие офисных замков`, `открыть замок официально`.

## Что уже проработано

- Первичная семантика под каждый сайт.
- Уникальное позиционирование, чтобы сайты не выглядели дублями друг друга.
- Title, Description, Keywords и Open Graph.
- Рекомендуемая структура H1-H3.
- SEO-тексты для основных блоков.
- FAQ для видимой страницы и JSON-LD.
- Рекомендации по alt/aria.
- Schema.org: `Locksmith`, `FAQPage`, `Review`, `OfferCatalog`.
- Базовое техническое on-page ТЗ.

## Следующий этап

- Расширить SEO через отдельные страницы услуг: вскрытие квартир, авто, сейфов, офисов, замена замков.
- Создать геостраницы под Москву, Санкт-Петербург, области и крупные города.
- Подключить Яндекс.Метрику и цели на звонки, формы и мессенджеры.
- Добавить реальные отзывы и не размечать тестовые отзывы как реальные после запуска.
- После добавления фотографий прописать `alt`, `width`, `height` и `loading="lazy"`.
